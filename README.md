# Delivery Hub

Standalone sister product to SPARK for commercial delivery workspaces.

## Design system

- Source baseline: `docs/design/SPARK_DESIGN_SOURCE.md`
- Hub design system: `docs/design/DELIVERY_HUB_DESIGN_SYSTEM.md`
- Immutable SPARK snapshot: `src/design-system-source/spark-679fe119/` (reference only)

## Development

```powershell
npm install
npm run dev
```

Open `http://localhost:8080/design-system` for the development-only design reference.

```powershell
npm run typecheck
npm run lint
npm test
npm run build
```
